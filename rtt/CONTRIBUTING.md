# Contributing to RTT

Thank you for your interest in contributing to the Relay Terminal Tool!

## Development Setup
```bash
git clone https://github.com/FlexNetOS/rtt-v1.git
cd rtt-v1/rtt-final
pip install -r requirements.txt
python auto/00-bootstrap.py
```

## Testing
```bash
python tests/validate.py
```

## Pull Request Process
1. Update documentation
2. Add tests
3. Follow code style
4. Get review approval
5. Merge to main

For questions or discussions: https://github.com/FlexNetOS/rtt-v1/discussions
